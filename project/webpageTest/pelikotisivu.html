About the game

The game was conceived during "Pelit ja Mobiilit" course in Metropolia in spring 2018.

Game requirements

The game requires:
  • Python3 interpreter
  • SQL database
  • Patience





The game took about 3 weeks to code and planning was about 2 weeks. A lot of changes were made to the original plan when we noticed some things were either too ambitious or too lackluster to add to the game.

Our method of working included about 3 times weekly face-to-face work and almost daily online work. Tools we used to manage work & communicate are discord and github.

The game is written in Python and takes advantage of a database written in mySQL (Python 3.4 & mysql ). The database contains roughly the following items:

Items 12
NPCs 13
Rooms 33
Extra items 28
Endings 5

Game Influences

As far as we know ourselves, this is what influenced the game:
  • Slouching towards Bedlam
  • Outlast
  • Shutter Island (2010)
  • Silent Hill
